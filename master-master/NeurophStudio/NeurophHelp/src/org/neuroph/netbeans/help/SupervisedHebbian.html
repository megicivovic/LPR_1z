SUPERVISED HEBBIAN NETWORK

Supervised Hebbian network is feedforward network that use Supervised Hebbian learning rule. Hebb's principle states that weight between two neurons will increase if the two neurons activate simultaneously,and it is reduced if they activate separately. It can be summarized as "cells that fire together, wire together". In the case of Supervised Hebbian Learning, the learning process is controlled (supervised) by Least Mean Squares rule.

Supervised Hebbian

To create and train Supervised Hebbian neural network using Neuroph Studio do the following:

  1. Create Neuroph project
  2. Create Supervised Hebbian network
  3. Create training set
  4. Train network
  5. Test trained network 

Step 1. To create Neuroph project click File > New Project.

 

Select Neuroph Project, click Next.

 

Enter project name and location, click Finish.

Project is created, now create neural network.

Step 2. Create Supervised Hebbian neural network. Click File > New File

 

Select project from Project drop-down menu, select Neural Network file type, click next.

 

Enter network name, select Supervised Hebbian network type, click next.

 

Enter number of neurons in input and output layer, choose Ramp transfer function as Createbutton.

 

 This will create the Supervised Hebbian

 Now we shall train this network to learn logical OR function. First we have to create the training set according to OR truth table, and we will use supervised learning method.

Step 3. In main menu click File >New File to open training set wizard.

 

Select Training set file type, then click next.

 

Enter training set name, choose type Supervised, number of inputs, Nextbutton.

 

Then create training set by entering training elements as input and desired output values of neurons in input and output layer. Use Add row button to add new elements or press Enter, and Finishbutton.

 

Step 4. Training network. To start network training procedure, in project window select training set Trainbutton.

In Set Learning parameters dialog use default learning parameters, and just click the Train button.

When the Total Net Error is zero, the training is complete.

Step 5. testing network. After the training is complete, you can test network by using Set Input button. This opens Set Network Input dialog in which you can enter input values for network separated with white space.

The result of network test is shown on picture below. Network learned logical OR function,so the value of output neuron is 1. Test to see how the network behaves for other inputs. You can also use Test button which will test the network with the selected training set.
