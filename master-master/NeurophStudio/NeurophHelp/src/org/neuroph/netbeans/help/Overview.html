NEUROPH FRAMEWORK OVERVIEW

The Neuroph Framework consists of the Java neural network library and integrated Neural Network & Java IDE based on NetBeans Platform - Neuroph Studio. It is the integtated environment to create and deploy neural networks to Java programs. Neuroph supports common neural network architectures, and it is very flexible so you can easily extend it to suit your specific needs.
